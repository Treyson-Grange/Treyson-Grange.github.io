Plant Tracker

This was my final project for CS 2410, Introduction to Event Driven Programming and GUI's.

Requirements included:

  • Use MVVM (Model view viewmodel architecture pattern)
  • App should use the navigation component
  • Data persists throughout subsequent app executions, meaning I had database integration
  • Custom App Icon
  • App is actually useful

Given these requirements I decided to create this app that would help me keep track of specific information that houseplants need
